Objetos

Para criar um objeto, basta definir uma variável e iniciar a definição de seu valor com chaves.


Constructor Functions
São responsáveis por construir novos objetos sempre que chamamos a mesma.
Keyword "new"
A palavra "new" é responsável por criar um novo objeto baseado na função construtora.
Keyword "this"
Faz referência ao próprio objeto costruido com a Constructor function.
Construtores nativos
Objetos, Funções, numeros, strings e outros tipos de dados são criados utilizando construtores. Esses construtores possue protótipo com propriedades e métodos que poderão ser acessados pelo tipo de dado.
const teste = Array.from(arrayLike) Converte um arrayLike em Array
const lista = Array.from(nodeList) Transforma uma nodeList em Array

Prototypes estão ligados a função construtora. Objetos criados a partir deste constructor não possuem prototype.

Caso um dos objetos instanciados tenha especificidades, como funções ou atributos diferentes, é possivel crialos no prototype, não afetando o construtor.

O Objeto criado utilizando o construtor possui acesso aos métodos e pripriedades do protótipo.


Host: Objetos do Host são aqueles implementados pelo próprio ambiente. Por exemplo no browser possuímos objetos do DOM, como DomList, HTMLCollection e outros. EM node.js os objetos do Host são diferentes, já que não estamos em um ambientedo browser.


String

length Propriedade com total de caracteres da string
charAt(n) Retorna o caractere de acordo com o index de (n)
concat Concatena as strings e retorna o resultado
includes(search, position) Procura pela string exata dentro de outra string. A procura é case sensitive
endsWith(search) e startsWith(search) Procura pela string exata dentro de outra string. A procura é case sensitive
slice(start, end) Corta a string de acordo com os valores de start e end
substring(start, end) Conta a string de acordo com os valores de start e end. Não funciona com valores negativos como o slice
lastIndexOf('teste') retorna a ultima ocorrência da string inserida
padStart(n, '-') e padEnd(n, '-') Aumenta o tamanho da string para o valor n. O preenchimento é feito com o valor especificado no segundo argumento ou espaços como default
repeat(n) Repete a string (n) vezes
replace(regexp|substr, newstr|function) Substitui parte da string. Utilizando um valor fixo, trocará apenas a primeira ocorrência
split(padrao) Divide a string de acordo com o padrão passado e retorna um Array
toLowerCase e toUpperCase Retorna a string em letras maiusculas ou minuscuals.
trim, trimStart e trimEnd Remove espaços em branco de ambos, inicio ou fim

Number

isNaN() e isInteger() verifica respectivamente se o valor é NaN e se é integral
parseFloat Retorna numeros a partir de uma string
parseInt Retorna numeros inteiros a partir de uma string
toFixed(n) Arredonda um numero com base no total de casas decimais n
toString(radix) Transforma um numero em uma string com base no Radix. Use 10 para sistema decimal.
toLocaleString(lang, options) Formata o número de acordo com a lingua e opções passadas

Math


É um objeto nativo que possui propriedades e métodos de expressões matemáticas comuns

abs() Retorna um valor absoluto, ou seja, transforma negativo em positivo
ceil() e floor() Arredonda o numero para cima, retornando sempre uma integral e floor para baixo
round() Arredonda um valor para o numero integral mais próximo.
max() Retorna o maior numero de uma lista de valores
min() Retorna o menor numero de uma lista de valores
random()

Array


Arrays armazenam uma coleção de elementos. Estes podem ser strings, arrays, booleans, numbers, functions objects e mais.


Array.isArray() Verifica se o valor passado é um array e retorna um valor booleano
[].length Retorna o tamanho do array

Métodos modificados


Além de retornarem um valor, modificam o Array original.

[].sort() Organiza pelo valor unicode
[].unshift() Adiciona elementos ao inicio do array e retorna o length dele.
[].push() Adiciona elementos ao final da array e retorna o length dele.
[].shift() Remove o primeiro elemento do array e retorna o mesmo.
[].pop() Remove o último elemento da array e retorna o mesmo.
[].reverse() Inverte os itens do array e retorna a nova array
[].copyWithin(alvo, iniciom, final) A partir do alvo, ele irá copiar a array começando do inicio até o final e vai preencher a mesma com essa cópia. Caso omita os valores do inicio e final, ele irá utilizar o inicio como 0 e o final o valor total do array.
[].fill(valor, inicio final) Preenche o array com o valor, do inivio até o fim
[].concat() O Array original não é modificado, apenas retorna um array modificado, onde os valores serão concatenados com o array fornecido.
[].includes(n) Verifica se o array possui o valor n e retorna true ou false
[].indexOf(n) Verifica se o array possui o valor n e retorna o index do primeiro n encotnrado
[].lastIndexOf(n) Verifica se o array possui o valor n e retorna o index do ultimo n encontrado
[].join(n) Junta todos os valores de um array e retorna uma string com eles. Fornecendo um valor n, ele será utilizado para a junção dos itens.
[].slice(inicio, final) Retorna os itens da array do inicio ao final especificado

Iterando Arrays


[].forEach() A função de callback é executada para cada item da array. Ela possui 3 argumentos, um representa o item atual, outro o index e o terceiro, o prório array.
[].map() Funciona da mesma forma que o forEach, porém ao invés de retornar undefined, retorna uma nova array com valores atualizados de acordo com o return de cada iteração.
[].reduce() Executa a função de callback para cada item do array. Um valor especial existe neste callback, chamado acumulador, que nada mais é que o retorno da iteração anterior.
