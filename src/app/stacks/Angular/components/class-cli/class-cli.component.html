Angular CLI

Comando Função
ng new nome Cria um novo projeto Angular
ng generate component Criar um novo componente Angular
ng new projeto --style=(sass/less/stylus ) Cria um projeto, setando um pré-processador default/td>
ng set defaults.styleExt scss/td> Seta um pré-processador em um projeto já criado/td>
ng lint Faz uma varredura para checar se seu código possui algum erro em potencial ou se segue os padrões e convenções da linguagem em uso
ng test Executa os testes criados nos arquivos spec
ng e2e Executa testes end to end
ng build Compila o projeto e realiza as tarefás que ô tornam apto a rodar em um servidor.
