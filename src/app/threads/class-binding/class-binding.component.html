Utilização Sintaxe
É possivel exibir um valor de uma variável no template inserindo-o entre chaves duplas: {{ site }}
É possivel a resolução de expressões dentro de uma interpolação: {{ 3 * 3 }}
É possivel invocar uma função dentro de uma interpolação: {{ 3 * 3 * getValor()}}
É possivel a interpretação de operadores lógicos dentro de uma interpolação: {{boolean1 || boolean2}}
